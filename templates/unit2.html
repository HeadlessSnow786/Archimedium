AP Calculus BC - Unit 2: Differentiation: Definition and Fundamental Properties

2.1 Definition of the Derivative

The derivative of a function measures how the function's output value changes as the input value changes. The derivative at a point can be defined as:

f'(x) = limh→0 (f(x+h) - f(x)) / h

Example: Find the derivative of f(x) = x².

  • f'(x) = limh→0 ((x+h)² - x²) / h
  • = limh→0 (x² + 2xh + h² - x²) / h
  • = limh→0 (2xh + h²) / h
  • = limh→0 2x + h
  • = 2x

2.2 The Power Rule

The power rule is a basic rule for differentiating functions of the form f(x) = xⁿ. If n is a real number, then:

f'(x) = n * x^(n-1)

Example: Find the derivative of f(x) = x³.

  • f'(x) = 3 * x^(3-1)
  • = 3x²

2.3 The Product Rule

For two functions u(x) and v(x), the product rule states:

(uv)' = u'v + uv'

Example: Find the derivative of f(x) = x² * sin(x).

  • Let u = x² and v = sin(x)
  • u' = 2x and v' = cos(x)
  • f'(x) = (x²)' * sin(x) + x² * (sin(x))'
  • = 2x * sin(x) + x² * cos(x)

2.4 The Quotient Rule

For two functions u(x) and v(x), the quotient rule states:

(u/v)' = (u'v - uv') / v²

Example: Find the derivative of f(x) = (x²) / (sin(x)).

  • Let u = x² and v = sin(x)
  • u' = 2x and v' = cos(x)
  • f'(x) = (2x * sin(x) - x² * cos(x)) / (sin(x))²

2.5 Derivatives of Trigonometric Functions

Derivatives of basic trigonometric functions include:

  • (sin(x))' = cos(x)
  • (cos(x))' = -sin(x)
  • (tan(x))' = sec²(x)

Example: Find the derivative of f(x) = tan(x).

  • f'(x) = sec²(x)

2.6 Chain Rule

The chain rule is used for differentiating composite functions. If a function y = f(u) and u = g(x), then the chain rule states:

dy/dx = (dy/du) * (du/dx)

Example: Find the derivative of f(x) = (3x² + 2)⁵.

  • Let u = 3x² + 2
  • f'(x) = 5 * (3x² + 2)⁴ * (6x)

2.7 Implicit Differentiation

Implicit differentiation is used when a function is not given explicitly. Differentiate all terms with respect to x, then solve for dy/dx.

Example: Find dy/dx if y³ + xy = 1.

  • d/dx (y³) + d/dx (xy) = d/dx (1)
  • 3y² * dy/dx + (x * dy/dx + y) = 0
  • dy/dx(3y² + x) = -y
  • dy/dx = -y / (3y² + x)
