AP Calculus BC Unit 5: Analytical Applications of Differentiation

1. Using the Mean Value Theorem (MVT)

The Mean Value Theorem states that if a function f is continuous on the closed interval [a, b] and differentiable on the open interval (a, b), then there exists at least one number c in (a, b) such that:

\[ f'(c) = \frac{f(b) - f(a)}{b - a} \]

Example:

Let f(x) = x2 on the interval [1, 3]. Verify the Mean Value Theorem and find the value of c.

First, compute the average rate of change:

\[ \frac{f(3) - f(1)}{3 - 1} = \frac{9 - 1}{2} = 4 \]

Next, find f'(x): f'(x) = 2x. Set this equal to 4 and solve for c:

2c = 4 → c = 2

Thus, c = 2 satisfies the Mean Value Theorem.

2. Connecting Increasing/Decreasing Behavior of function and its Derivative

A function f is increasing on an interval if f'(x) > 0 for all x in that interval. It is decreasing if f'(x) < 0 for all x in that interval.

Example:

Given f(x) = x3 − 3x + 1, determine where the function is increasing and decreasing.

First, find f'(x): f'(x) = 3x2 − 3.

Solve for critical points: 3x2 − 3 = 0 → x = ±1.

Test intervals around critical points:

  • (-∞, -1): choose x = -2 → f'(-2) = 3(-2)2 − 3 = 9 > 0 (increasing)
  • (-1, 1): choose x = 0 → f'(0) = -3 < 0 (decreasing)
  • (1, ∞): choose x = 2 → f'(2) = 3(2)2 − 3 = 9 > 0 (increasing)

Therefore, the function is increasing on (-∞, -1) and (1, ∞), and decreasing on (-1, 1).

3. Using the First Derivative Test to Determine Local Extrema

The First Derivative Test helps identify local maxima and minima. If f'(x) changes from positive to negative at c, then f(c) is a local maximum. If f'(x) changes from negative to positive at c, then f(c) is a local minimum.

Example:

Find the local extrema of f(x) = x3 − 3x2 + 2.

First, find f'(x): f'(x) = 3x2 − 6x.

Set f'(x) = 0 and solve for x: 3x(x − 2) = 0 → x = 0, 2.

Test intervals around critical points:

  • (-∞, 0): choose x = -1 → f'(-1) = 9 > 0
  • (0, 2): choose x = 1 → f'(1) = -3 < 0
  • (2, ∞): choose x = 3 → f'(3) = 9 > 0

Since f'(x) changes from positive to negative at x = 0, there is a local maximum at x = 0.

Since f'(x) changes from negative to positive at x = 2, there is a local minimum at x = 2.

4. Using the Second Derivative Test to Determine Concavity and Inflection Points

The Second Derivative Test helps determine concavity and inflection points. If f''(x) > 0 on an interval, f is concave up. If f''(x) < 0 on an interval, f is concave down. Points where concavity changes are called inflection points.

Example:

Determine the concavity and inflection points of f(x) = x3 − 6x2 + 12x.

First, find f''(x): f''(x) = 6x − 12.

Set f''(x) = 0 and solve for x: 6x − 12 = 0 → x = 2.

Test intervals around x = 2:

  • (-∞, 2): choose x = 0 → f''(0) = -12 < 0 (concave down)
  • (2, ∞): choose x = 3 → f''(3) = 6 > 0 (concave up)

There is an inflection point at x = 2 where concavity changes from down to up.

5. Sketching Graphs of Functions Using First and Second Derivatives

To sketch the graph of a function:

  • Find critical points and determine where the function is increasing or decreasing.
  • Use the first derivative test to identify local maxima and minima.
  • Find points of inflection and determine the intervals of concavity using the second derivative test.
